## ID & Naming Conventions

### Document IDs
```
<TYPE>-<NNN>_<slug>

Types: SDP, SRS, SDD, VVP, VVR, TM, RM, SOUP, CM, REL, ADR
NNN  : 3-digit sequential number (001, 002, ...)
slug : lowercase-hyphenated description
```

### Inline IDs (used within documents)
```
REQ-<layer><NNN>    Requirements   REQ-B001, REQ-N001, REQ-SC001
```
